# Mohamed Saber Abbas Elsayed — Portfolio

Personal portfolio website and curriculum vitae for **Mohamed Saber Abbas Elsayed**, Computer & Systems Engineering student at the Faculty of Engineering, Alexandria University, and Frontend Engineering Trainee at Capgemini.

## Stack

- Semantic HTML5
- Modern CSS with custom properties (dark navy + gold + glacial blue glassmorphism)
- Vanilla JavaScript (no frameworks, no build step)
- Canvas API particle field, IntersectionObserver scroll reveals, typewriter effect

## Layout

```
portfolio/
├── index.html              Single-page portfolio
├── css/styles.css          Theme + glassmorphism system
├── js/main.js              Particles, cursor, reveals, typewriter
├── assets/
│   ├── favicon.svg
│   ├── profile.jpg
│   └── profile-sm.jpg
└── cv/
    ├── cv.md               Markdown CV source
    ├── cv.html             Print-styled CV
    └── Mohamed_Saber_Abbas_Elsayed_CV.pdf
```

## Local development

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080/`.
